# DanceStudio — Веб-интерфейс танцевальной студии

Курсовой проект по дисциплине «Информационные технологии и программирование».  
Разработка клиентского веб-приложения для танцевальной студии с фильтрацией, избранным, переключением тем и формой записи.

**🔗 GitHub Pages:** [aiilluj.github.io/Dance-studio/](https://aiilluj.github.io/Dance-studio/)

---
## 📋 Техническое задание

### Общие требования
Назначение разрабатываемого программного продукта — создание современного, информативного и визуально привлекательного веб-интерфейса для танцевальной студии DanceStudio. Сайт предоставляет потенциальным клиентам возможность ознакомиться с танцевальными направлениями, информацией о преподавателях, преимуществах студии, а также записаться на пробное занятие. Продукт является клиентским веб-приложением (frontend-only) и не требует серверной инфраструктуры: все данные хранятся локально либо в коде приложения.

### Функциональные требования

**Обязательные:**
- Адаптивное горизонтальное меню с плавной прокруткой к секциям: «Главная», «О студии», «Направления», «Преподаватели», «Контакты»
- Главный экран: крупное фоновое изображение, название студии, приветственный текст, кнопка «Записаться на пробный урок»
- Секция «О студии»: текстовое описание, сетка преимуществ, блок статистики
- Сетка карточек танцевальных направлений (не менее 6): название, описание, цена, изображение
- Форма записи на занятие с полями «Имя», «Телефон», выпадающим списком направлений и клиентской валидацией
- Переключение светлой и тёмной темы с сохранением в localStorage

**Желательные:**
- Фильтрация карточек направлений по категориям («Для взрослых», «Для детей»)
- Панель избранного с сохранением в localStorage
- Текстовый поиск по названию и описанию направления

### Технические требования
Сайт функционирует на любом современном компьютере или мобильном устройстве. Минимальные технические характеристики: процессор от 1,6 ГГц, ОЗУ от 512 МБ. Рекомендуемое разрешение экрана — от 1024×768 пикселей.

### Программные требования
Сайт работает в любых ОС (Windows, macOS, Linux, iOS, Android). Требуются современные браузеры: Google Chrome 90+, Mozilla Firefox 88+, Microsoft Edge 90+, Safari 14+. Разработка ведётся в VS Code. Для управления версиями используются Git и GitHub. Готовый проект публикуется на GitHub Pages.

## 📁 Структура проекта

```text
Dance-studio/
├── index.html                  # Главная страница
├── style.css                   # Стили и CSS-переменные
├── main.js                     # Точка входа
├── data/
│   ├── directions.js           # Статические данные о направлениях
│   └── trainers.js             # Статические данные о преподавателях
├── classes/
│   ├── DanceDirection.js       # Класс-сущность направления
│   ├── FavoritesManager.js     # Управление избранным + localStorage
│   ├── ThemeManager.js         # Управление темой + localStorage
│   ├── FormValidator.js        # Валидация формы
│   └── AppController.js        # Главный контроллер приложения
└── images/                     # Изображения (hero, directions, trainers)
```
