# DanceStudio — Веб-интерфейс танцевальной студии

Курсовой проект по дисциплине «Информационные технологии и программирование».  
Разработка клиентского веб-приложения для танцевальной студии с фильтрацией, избранным, переключением тем и формой записи.

**🔗 GitHub Pages:** [aiilluj.github.io/Dance-studio/](https://aiilluj.github.io/Dance-studio/)

---

## 📁 Структура проекта

```text
Dance-studio/
├── index.html                  # Главная страница
├── style.css                   # Стили и CSS-переменные
├── main.js                     # Точка входа
├── data/
│   ├── directions.js           # Статические данные о направлениях
│   └── trainers.js             # Статические данные о преподавателях
├── classes/
│   ├── DanceDirection.js       # Класс-сущность направления
│   ├── FavoritesManager.js     # Управление избранным + localStorage
│   ├── ThemeManager.js         # Управление темой + localStorage
│   ├── FormValidator.js        # Валидация формы
│   └── AppController.js        # Главный контроллер приложения
└── images/                     # Изображения (hero, directions, trainers)
```
## 🚀 Установка и запуск

### Локальный запуск

1. **Клонируйте репозиторий:**

   ```bash
   git clone https://github.com/aiilluj/Dance-studio.git
2. **Откройте папку проекта** в VS Code.

3. **Запустите** `Live Server` (расширение) или просто откройте файл `index.html` в браузере.
